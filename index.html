Installing Visual Studio Code

Now that you've downloaded the VS Code installer, let's go through the installation process.

1

Run the installer

Locate the downloaded file (usually in your Downloads folder) and double-click it

2

Accept the agreement

Read and accept the license agreement

3

Choose installation location

Select where you want to install VS Code or use the default location

4

Select additional tasks

Choose options like "Add to PATH" (recommended) and "Create desktop icon"

?
5

Complete installation

Click "Install" and wait for the process to complete

Installing Python

Now that you've downloaded Python, let's install it on your Windows 11 system.

1

Run the installer

Locate the downloaded Python installer and double-click it

2

Enable important options

Check "Add Python to PATH" (very important!)

!
3

Choose installation type

Select "Customize installation" for more options

4

Select optional features

Ensure "pip" and "Python test suite" are selected

5

Advanced options

Set your installation location and click "Install"

Python installation with PATH option highlighted

Common Issue

If you forget to check "Add Python to PATH," you can fix it later by modifying your system's environment variables or reinstalling Python.

Verify installation in Command Prompt 
python --version

Setting up Python in VS Code

Finally, let's configure VS Code to work with Python so you can start coding.

1

Open VS Code

Launch Visual Studio Code from the Start menu or desktop

2

Install Python extension

Open the Extensions view (Ctrl+Shift+X) and search for "Python"

3

Install Microsoft's Python extension

Click "Install" on the official Microsoft Python extension

4

Create a Python file

Create a new file with a .py extension (e.g., hello.py)

5

Select Python interpreter

VS Code will prompt you to select a Python interpreter. Choose the one you installed.
