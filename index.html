Visual Studio Codeni o'rnatish

Endi siz VS Code o'rnatuvchisini yuklab olganingizdan so'ng, o'rnatish jarayoniga o'tamiz. Yuklab olingan .exe faylni shunchaki oching.

1

Dasturni oching

Yuklab olingan faylni toping (odatda Yuklashlar jildida bo'ladi) va uni ikki marta bosing

2

Shartnomani qabul qiling

Litsenziya shartnomasini o'qing va qabul qiling aks xolda dasturni o'rnatish uchun Keyingi (Next) tugmasi bosilmaydi!

3

Dastur o'rnatilishi uchun joylashuv belgilang!

VS kodini o'rnatmoqchi bo'lgan joyni tanlang yoki standart joylashuvdan foydalaning. Standart C: diskga o'rnatiladi yoki istalgan diskga o'rnatishingiz mumkin.

4

Qo'shimcha vazifalarni tanlang

VS Codeda Python-ni sozlash

Va nihoyat, kodlashni boshlashingiz uchun VS Codeni Python bilan ishlash uchun sozlaymiz.

1

Open VS Code

Launch Visual Studio Code from the Start menu or desktop

2

Python kengaytmasini o'rnating

VS Codedan Extensions bo'limiga o'ting, qisqa klaviratura kombinatsiyasi (Ctrl+Shift+X) qidiruvga "Python" atamasini yozing

3

Microsoft-ning Python kengaytmasini o'rnating

Birinchida chiqgan Python Extensions usiga bosing va "Install" tugmasini bosing hamda biroz kuting.

4

Python faylini yarating

Endi File bo'limidan New File ni tanlang chiqgan oynadan Python fileni tanlang.

5

Faylni saqlang

File bo'limidan Save ni tanlang va fayllardan istalgan joyga kodingizni saqlang(Eslatma: Kodlar uchun alohida papka yaratib qo'yib usha papkada saqlashingiz mumkin bu kampyuteringizni tartibli ishlashiga yordam beradi.).
